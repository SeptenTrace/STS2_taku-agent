# STS2 Taku Agent - Install Guide

This mod package includes:
- `taku_agent.pck`
- `taku_agent.dll`

Both files are required.

## Windows (Steam)
1. Open your game install folder:
   - `...\Steam\steamapps\common\Slay the Spire 2\`
2. Create `mods` folder if it does not exist:
   - `...\Slay the Spire 2\mods\`
3. Copy `taku_agent.pck` and `taku_agent.dll` into `mods`.
4. Launch the game and enable mod loading when prompted.

## macOS (Steam)
1. Open:
   - `~/Library/Application Support/Steam/steamapps/common/Slay the Spire 2/SlayTheSpire2.app/Contents/MacOS/`
2. Create `mods` folder if it does not exist.
3. Copy `taku_agent.pck` and `taku_agent.dll` into `mods`.
4. Launch the game and enable mod loading when prompted.

## Troubleshooting
- Confirm both files share the same base name:
  - `taku_agent.pck`
  - `taku_agent.dll`
- If you updated from an older version, fully exit the game and relaunch.
