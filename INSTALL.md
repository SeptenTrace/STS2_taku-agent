# STS2 Taku Agent - Install Guide

This mod package includes:
- `mod_manifest.json`
- `taku_agent.pck`
- `taku_agent.dll`

All three files are required and should be kept in the same mod subdirectory.

## Windows (Steam)
1. Open your game install folder:
   - `...\Steam\steamapps\common\Slay the Spire 2\`
2. Create `mods` folder if it does not exist:
   - `...\Slay the Spire 2\mods\`
3. Create a subdirectory such as `mods\\taku_agent\\`.
4. Copy `mod_manifest.json`, `taku_agent.pck`, and `taku_agent.dll` into that subdirectory.
5. Launch the game and enable mod loading when prompted.

## macOS (Steam)
1. Open:
   - `~/Library/Application Support/Steam/steamapps/common/Slay the Spire 2/SlayTheSpire2.app/Contents/MacOS/`
2. Create `mods` folder if it does not exist.
3. Create `mods/taku_agent/`.
4. Copy `mod_manifest.json`, `taku_agent.pck`, and `taku_agent.dll` into `mods/taku_agent/`.
5. Launch the game and enable mod loading when prompted.

## Troubleshooting
- Confirm the mod directory contains:
  - `mod_manifest.json`
  - `taku_agent.pck`
  - `taku_agent.dll`
- If you updated from an older version, fully exit the game and relaunch.
